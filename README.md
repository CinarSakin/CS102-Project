# CS102 PROJECT | Dungeonfall ⚔️

Dungeonfall is a 2D rogue-like dungeon crawler where the player goes through procedurally generated levels while gathering items in rooms and defeating bosses at the end of each level.

## Features

🗺️ **Level Generation:** Level is generated randomly using Binary Space Partitioning. \
🎮 **Game Modes:** There is a **Standard** mode which ends after defeating the boss and an **Infinite** mode. \
🏆 **Global Ranking:** Compare the points you got in Infinite mode with others via MongoDB integrated leaderboard. \
💾 **Save System:** You can save your progress in 6 save slots locally in JSON files. \
⚙️ **Customization:** Adjustable keybinds, UI scaling, and attack type.

## Dependencies 🛠️

* **Language & UI:** Java 21, JavaFX 23.
* **Build Tool:** Maven.
* **Database:** MongoDB.
